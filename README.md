# Youtube-cli
> A Youtube command line tool for Linux.

---

# Installation Instructions

### Prerequisites:
1. NodeJS & NPM

Clone this repository:
```
$ git clone https://www.github.com/r-dog/youtube-cli
```

In the Linux folder, run:
```
$ npm link
```
